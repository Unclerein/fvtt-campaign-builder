# Tab Visibility Settings

Tab Visibility Settings allow you to control which tabs are visible for each content type in the Campaign Builder. This helps streamline your interface by hiding tabs you don't use.

::: info
Description tabs are always shown and cannot be hidden.
:::

## Available Options

Tab visibility can be controlled independently for different content types and their tabs:

### Settings
- Journals

### Characters
- Journals
- Locations
- Organizations
- PCs
- Sessions
- Foundry general
- Actors

### Locations
- Journals
- Characters
- Organizations
- PCs
- Sessions
- Foundry general
- Scenes

### Organizations
- Journals
- Characters
- Locations
- PCs
- Sessions
- Foundry general

### PCs
- Journals
- Characters
- Locations
- Organizations
- Foundry general

### Campaigns
- Journals
- PCs
- Lore
- Ideas
- To-Dos
- Story Webs

### Arcs
- Journals
- Lore
- Vignettes
- Locations
- Participants
- Monsters
- Ideas
- Story Webs

### Sessions
- Lore
- Vignettes
- Locations
- NPCs
- Monsters
- Magic Items
- PCs
- Story Webs
