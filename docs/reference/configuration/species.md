# Species List Menu 

![Species Settings](/assets/images/species-settings.webp)

This lets you manage the list of species in your worlds.  The default species are from the D&D 5E SRD.  This list controls what Species are available in the Species list on the Character details page.  The descriptions are only important for facilitating more accurate descriptions/images when doing AI generation [[Advanced Feature]].
