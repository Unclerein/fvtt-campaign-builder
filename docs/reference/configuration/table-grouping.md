# Table Grouping Settings

The Table Grouping sub-menu allows you to control which tables use [grouping functionality](/reference/navigation/tables#groups) throughout Campaign Builder. Grouped tables let you organize items into custom groups for better organization.

## Available Options

Table grouping can be enabled or disabled independently for different content types and their tabs:

### Campaigns
- PCs
- Lore
- Ideas
- To-Dos

### Arcs
- Lore
- Vignettes
- Locations
- Participants
- Monsters
- Ideas

### Sessions
- Lore
- Vignettes
- Locations
- NPCs
- Monsters
- Magic Items

## How It Works

When table grouping is enabled for a tab:
- A folder icon button appears in the table header, allowing you to create new groups
- Items can be organized into custom groups
- Drag and drop can move items between groups
- Groups can be renamed or deleted

When table grouping is disabled:
- The table displays as a simple flat list
- No grouping controls are shown
- Existing groups are hidden (but not deleted - they'll reappear if grouping is re-enabled)

::: info TIP
If you don't need the organization that grouping provides, disabling it can simplify the interface and save screen space.
:::

- **Magic Items** - Group magic items in Session details
