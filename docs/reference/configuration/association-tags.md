# Association Tags

Association tags let you connect Entry tags to Foundry Actors and Scenes. When an Character/Location has a tag that's associated with an Actor/Scene, that Foundry document becomes automatically available in the entry's Actors or Scenes tab just as if it had been manually attached.

This way if you have common Actors/Scenes that you reuse across Characters/Locations (ex. a "commoner" actor for all your generic townspeople NPCs), you can simply apply the tag instead of having to individually connect them.

## Overview

The Association Tags configuration dialog has two tabs:
- **Characters tab**: Configure tags that link to Foundry Actors
- **Locations tab**: Configure tags that link to Foundry Scenes

Each association tag has:
- **Name**: The tag name
- **Color**: A color for visual identification in tag displays
- **Actor/Scene**: The Foundry document to associate with this tag

## Managing Associations

- **Edit**: Click on any field (name, color, or actor/scene) to modify it
- **Delete**: Click the trash icon to remove an association tag
- **Reset**: Click the Reset button to undo all changes since opening the dialog
- **Save**: Click Save Changes to persist your modifications

Changes don't take effect until you click Save Changes.
