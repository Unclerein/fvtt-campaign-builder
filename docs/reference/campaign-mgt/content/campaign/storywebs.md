# Story Webs Tab
![Campaign Story Webs Tab](/assets/images/campaign-storywebs-tab.webp)

The Story Webs tab allows you to associate [Story Webs] with the overall Campaign to make them easier to manage.  

> [!Note]
> The entire list of Story Webs can be found in the Story Webs folder on the Campaign in the Campaign Directory.  But that folder isn't well-suited for organization, so if you want to say "This Story Web is for the overall Campaign" (as opposed to others that are stored in the Campaign in the directory but are really only relevant to a specific Arc or Session), this tab is where you manage that.

This tab provides:

## Adding Story Webs
Click "Add Story Web" to associate a Story Web with this Campaign.  You will be prompted to search for an existing Story Web.  

You can also drag/drop a Story Web from the Campaign Directory.

> [!INFO]
> A single Story Web can be associated with any number of Arcs and/or Sessions, as well as the Campaign.

## The Story Web list
This list contains the selected Story Webs.  

Click the name of a Story Web to open it (hold Control to open in a new tab).

### Moveable
You can reorder the story web list by dragging the handle on the left side of each row to move rows up and down.

### Story Web Actions
The actions column lets you manage the Story Web entries:
  - **Delete** - Delete the Story Web.  This removes it from the Campaign list, but DOES NOT delete the Story Web itself - it remains in the Campaign Directory for future use.
