# Campaign Timeline Tab
![Campaign Timeline Tab](/assets/images/campaign-timeline-tab.webp)

The Timeline tab shows calendar events from Calendaria relevant to your [^Campaign]. This helps you track events that affect your campaign's story and characters.

::: info
Timelines require the Calendaria module to be installed and active. See the main [Timelines] documentation for more details on this, as well as general usage of Timelines.
:::

## Filtering Events

You can set filters, as well as the viewable window to control what the timeline view for this Campaign should be. Those settings are content-specific (i.e. this Campaign) and are saved.

### Reference This Campaign

By default, this option is enabled and filters the timeline to show only events that reference this Campaign in their content (using `@UUID` links). This shows events specifically tagged as relevant to this campaign.

Disable this option to see all calendar events instead.

### Include Nested Content

When enabled (requires "Reference This Campaign" be on), the timeline shows events that reference:

- All [^Arcs] in this campaign
- All [^Sessions] in this campaign

This gives you a comprehensive view of everything connected to your campaign's story.
