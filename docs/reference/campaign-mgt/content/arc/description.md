# Arc Description Tab
![Arc Description Tab](/assets/images/arc-description-tab.webp)

The description tab has two components:
1. Image
2. Arc description

## Image
See [Image] documentation.

## Arc description

Use the main editor box to capture:
- What this Arc is about.
- How and when it starts.
- Major events, threats, or conflicts you expect.
- How it might end.

See [Editors] for more information on the capabilities of editor boxes.
